This document describes the design system's current aesthetic and functional properties.

### Color Palette
The theme uses a `dark` color mode, featuring a `primary_color_hex` of `#2d5a27`, a `secondary_color_hex` of `#a64d32`, a `tertiary_color_hex` of `#f1c40f`, and a `neutral_color_hex` of `#2b2624`. These colors define the brand's visual identity across all components.

### Typography
- **Headlines:** Set in `spaceGrotesk`, providing a distinct and modern feel.
- **Body Text:** Uses `manrope`, ensuring readability and consistency for longer content.
- **Labels:** Also in `manrope`, for clear and legible interactive elements.

### Shape and Form
- **Roundedness:** A value of `2` indicates a moderate level of roundedness for UI elements, balancing modern aesthetics with user-friendliness.

### Spacing
- **Density:** A `spacing` value of `2` provides a normal level of whitespace, creating a balanced and comfortable visual density across the interface.
